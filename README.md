# TestIPC
该程序演示几种IPC方式，参考《Android开发艺术探索》；

##程序说明
该程序实现了通过Messenger、AIDL、Binder连接池、ContentProvider、Socket五种方式实现IPC的功能；

程序只是简单实现了功能，可以用来参考学习；

##总结
选择最合适的IPC方式
